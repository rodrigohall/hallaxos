# Hall Eventos

Cérebro operacional da Hall Eventos (Dourados-MS) — locação de itens para festas e coquetelaria.

Projeto conceitualmente independente do HallaxOS (o outro sistema neste monorepo, em `apps/` e
`docs/`, feito para uma empresa de veículos). Nenhum código ou dado é compartilhado entre os
dois — só o método: arquitetura definida antes do código, documentação viva.
